## 16.0 形度是什么？

### 16.0.1 从格度到形度：超越离散

在Chapter 14中，我们学习了**格度（Grid-Degree）**——时空的最小离散单元。

但格度不是死板的方格纸，它会**振动**、**涨落**、**模糊**。

**形度（Shape-Degree）**就是描述**格度振动的模糊度数**。

**430的类比**：
> "就好像你看水面一样，当水波出现时自然就会呈现出来不同的倒影。"

- **平静水面**（格度清晰）：倒影清晰，格度边界明确
- **波纹水面**（格度模糊）：倒影扭曲，格度边界模糊
- **汹涌波涛**（格度剧烈振动）：倒影破碎，格度高度整合

**核心定义**：

**Definition 16.0** (Shape-Degree):
**形度（Shape-Degree）**是格度的细粒度整合度，描述格度振动产生的模糊层次。

$$\mathcal{D}_{\text{shape}}(\mathcal{G}) = \int_{\mathcal{G}} \rho_{\text{fuzzy}}(g, \sigma) \, dg$$

其中：
- $\mathcal{G}$：格度空间（见Chapter 14）
- $\rho_{\text{fuzzy}}(g, \sigma)$：格度g在尺度σ下的模糊密度
- 积分表示对所有格度单元的整合

**关键洞察**：
形度不是几何形状，而是**格度如何被整合的度量**。

## 16.D.2 形度函数与形度密度

## 16.D.3 形度熵：比超熵更高级的概念

### 16.D.3.1 形度熵的定义

**Definition 16.D.4** (Shape-Degree Entropy, QGE):

$$\mathcal{S}_{\text{QGE}} = -\sum_X p(X) \log \phi(X)$$

- $X$：可能的量子概形状态

**与信息熵的区别**：
