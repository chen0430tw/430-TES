# Chapter 8: ACT高级批判性思维
## Advanced Critical Thinking: Deep Analysis and Logical Reasoning

*批判性思维不是批评一切,而是深入分析一切。ACT将批判性思维提升到高级水平:不仅质疑结论,更审查推理过程;不仅识别谬误,更构建严密论证;不仅破坏,更建设。*

## 8.1 ACT的理论基础

**Definition 8.1** (Advanced Critical Thinking):
ACT是系统化的高阶认知技能,包括深度分析、逻辑推理、证据评估和论证构建。

**核心能力**:
1. **论证分析**: 识别前提、结论、推理链
2. **谬误检测**: 识别各类逻辑谬误
3. **证据评估**: 评价证据的质量和相关性
4. **论证构建**: 构建严密的推理链条

**Theorem 8.1** (Critical Thinking Hierarchy):
批判性思维能力分层:

$$\text{识别} < \text{分析} < \text{评估} < \text{创造}$$

## 8.2 深度分析与逻辑推理

**逻辑推理类型**:

**1. 演绎推理**: 前提真→结论必然真
**2. 归纳推理**: 前提真→结论可能真
**3. 溯因推理**: 推测最佳解释

**Theorem 8.2** (Validity vs Soundness):
论证有效性≠论证合理性。

*Proof*:
1. 有效论证:结论逻辑上跟随前提
2. 合理论证:有效+前提为真
3. 例:"所有unicorns会飞,这是unicorn,所以它会飞"(有效但不合理)
4. Therefore, 必须检查前提真实性 ∎

**Practice 8.1** (Argument Analysis):
分析以下论证:
"AI会取代所有工作。你的工作涉及认知任务。因此AI会取代你的工作。"
1. 识别前提和结论
2. 评估推理有效性
3. 挑战前提
4. 构建反论证
